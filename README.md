# ✳️ Front End Test Project

    Coderockr test project.

## ✔️ Tech Stack

- Reactjs
- Tailwindcss
  - Used to style the project faster
- Axios
  - Used to fetch api GET method
- Typescript
  - Used to add static typing to the code
- React Testing Library

## ✔️Install

```
yarn install
```

### ✔️ Develop

```
yarn start
```

## ✔️Build

```
yarn build
```
